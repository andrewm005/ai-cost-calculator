Frequently asked questions

How much does Command A cost per 1M tokens?

Command A costs $2.50 per 1M input tokens and $10.00 per 1M output tokens on OpenRouter. A typical chat run (1,000 input, 500 output) costs about $0.0075; a long-context job (50,000 input, 5,000 output) costs about $0.175.

What is the context window for Command A?

The context window for Command A is 256,000 tokens, suitable for long documents and extended agent loops. Anything longer has to be chunked or summarized before it fits.

Does Command A support tool use / function calling?

OpenRouter does not publish a tool-use flag for Command A on this listing, so assume no and validate in your own integration. If the upstream feed adds the flag, the next six-hourly refresh will surface it here.

Is Command A cheaper than North Mini Code (free)?

No — Command A is more expensive than North Mini Code (free) on input by about 100% ($2.50 vs $0.00 per 1M). Output is $10.00 per 1M versus $0.00 for North Mini Code (free). See the North Mini Code (free) model page for the full price table and workload examples.

How much does it cost to run Command A for coding agent?

A typical coding agent run (8,000 input + 4,000 output + 5 tool calls) on Command A costs about $0.0600 per call. At 10,000 runs/month that is roughly $600.00. The bulk of that is output at $10.00 per 1M tokens.
