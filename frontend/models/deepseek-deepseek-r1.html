About R1

R1 is a mid-tier model in the DeepSeek lineup, priced at $0.70 per 1M input tokens and $2.50 per 1M output tokens. It is the kind of model most teams end up defaulting to once they have measured their workloads: cheap enough to run at moderate volume, capable enough that the answer quality does not require a flagship. DeepSeek generally publishes a budget option below this model and a flagship above it, and R1 is the middle step in that ladder.

At $0.70 per 1M input tokens and $2.50 per 1M output tokens, the cost of a single call is dominated by output unless the workload is very output-heavy or very input-heavy in unusual ways. Output is roughly 3.6x the input cost, which is the typical shape for chat-tuned models where generation is the expensive direction. The context window is about 163K tokens, which covers most single-turn chat and short retrieval tasks but requires chunking for long documents. DeepSeek's pricing for this model is published through OpenRouter and refreshed every six hours; this page reads the live values, so the numbers here match what you would see if you called the OpenRouter models endpoint directly.

Frequently asked questions

How much does R1 cost per 1M tokens?

R1 costs $0.70 per 1M input tokens and $2.50 per 1M output tokens on OpenRouter. A typical chat run (1,000 input, 500 output) costs about $0.0019; a long-context job (50,000 input, 5,000 output) costs about $0.048.

What is the context window for R1?

The context window for R1 is 163,840 tokens, which covers most single-turn chat and short retrieval tasks. Anything longer has to be chunked or summarized before it fits.

Does R1 support tool use / function calling?

OpenRouter does not publish a tool-use flag for R1 on this listing, so assume no and validate in your own integration. If the upstream feed adds the flag, the next six-hourly refresh will surface it here.

Is R1 cheaper than DeepSeek V3.1?

No — R1 is more expensive than DeepSeek V3.1 on input by about 70% ($0.70 vs $0.21 per 1M). Output is $2.50 per 1M versus $0.79 for DeepSeek V3.1. See the DeepSeek V3.1 model page for the full price table and workload examples.

How much does it cost to run R1 for complex reasoning?

A typical complex reasoning run (2,000 input + 4,000 output) on R1 costs about $0.0114 per call. At 10,000 runs/month that is roughly $114.00. The bulk of that is output at $2.50 per 1M tokens.
