About Nova Lite 1.0

Nova Lite 1.0 is a budget-tier model on OpenRouter, priced at $0.06 per 1M input tokens and $0.24 per 1M output tokens. It is the right pick for high-volume work where the cost per call matters more than the absolute best answer quality: bulk classification, extraction pipelines, simple chat agents, and the kind of background work that fires thousands of times a day. It is also a sensible default for prototypes where the team has not yet decided which model they want to standardize on, since the bill will be small even if the volume is high.

At $0.06 per 1M input tokens and $0.24 per 1M output tokens, the cost of a single call is dominated by output unless the workload is very output-heavy or very input-heavy in unusual ways. Output is priced noticeably above input (4.0x), reflecting the typical chat-shape where the model spends more compute on generation than on reading the prompt. The context window is about 300K tokens, suitable for long documents and extended agent loops without having to chunk the input. Amazon's pricing for this model is published through OpenRouter and refreshed every six hours; this page reads the live values, so the numbers here match what you would see if you called the OpenRouter models endpoint directly.

Frequently asked questions

How much does Nova Lite 1.0 cost per 1M tokens?

Nova Lite 1.0 costs $0.06 per 1M input tokens and $0.24 per 1M output tokens on OpenRouter. A typical chat run (1,000 input, 500 output) costs about $0.0002; a long-context job (50,000 input, 5,000 output) costs about $0.004.

What is the context window for Nova Lite 1.0?

The context window for Nova Lite 1.0 is 300,000 tokens, suitable for long documents and extended agent loops. Anything longer has to be chunked or summarized before it fits.

Does Nova Lite 1.0 support tool use / function calling?

OpenRouter does not publish a tool-use flag for Nova Lite 1.0 on this listing, so assume no and validate in your own integration. If the upstream feed adds the flag, the next six-hourly refresh will surface it here.

Is Nova Lite 1.0 cheaper than Llama 3.2 1B Instruct?

No — Nova Lite 1.0 is more expensive than Llama 3.2 1B Instruct on input by about 55% ($0.06 vs $0.03 per 1M). Output is $0.24 per 1M versus $0.20 for Llama 3.2 1B Instruct. See the Llama 3.2 1B Instruct model page for the full price table and workload examples.

How much does it cost to run Nova Lite 1.0 for bulk extraction?

A typical bulk extraction run (500 input + 200 output) on Nova Lite 1.0 costs about $0.000078 per call. At 10,000 runs/month that is roughly $0.7800. The bulk of that is output at $0.2400 per 1M tokens.
