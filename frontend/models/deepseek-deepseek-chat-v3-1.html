About DeepSeek V3.1

DeepSeek V3.1 is a budget-tier model on OpenRouter, priced at $0.21 per 1M input tokens and $0.79 per 1M output tokens. It is the right pick for high-volume work where the cost per call matters more than the absolute best answer quality: bulk classification, extraction pipelines, simple chat agents, and the kind of background work that fires thousands of times a day. It is also a sensible default for prototypes where the team has not yet decided which model they want to standardize on, since the bill will be small even if the volume is high.

At $0.21 per 1M input tokens and $0.79 per 1M output tokens, the cost of a single call is dominated by output unless the workload is very output-heavy or very input-heavy in unusual ways. Output is roughly 3.8x the input cost, which is the typical shape for chat-tuned models where generation is the expensive direction. The context window is about 163K tokens, which covers most single-turn chat and short retrieval tasks but requires chunking for long documents. DeepSeek's pricing for this model is published through OpenRouter and refreshed every six hours; this page reads the live values, so the numbers here match what you would see if you called the OpenRouter models endpoint directly.
