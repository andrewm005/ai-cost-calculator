About Nova Pro 1.0

Nova Pro 1.0 is a mid-tier model in the Amazon lineup, priced at $0.80 per 1M input tokens and $3.20 per 1M output tokens. It is the kind of model most teams end up defaulting to once they have measured their workloads: cheap enough to run at moderate volume, capable enough that the answer quality does not require a flagship. Amazon generally publishes a budget option below this model and a flagship above it, and Nova Pro 1.0 is the middle step in that ladder.

At $0.80 per 1M input tokens and $3.20 per 1M output tokens, the cost of a single call is dominated by output unless the workload is very output-heavy or very input-heavy in unusual ways. Output is priced noticeably above input (4.0x), reflecting the typical chat-shape where the model spends more compute on generation than on reading the prompt. The context window is about 300K tokens, suitable for long documents and extended agent loops without having to chunk the input. Amazon's pricing for this model is published through OpenRouter and refreshed every six hours; this page reads the live values, so the numbers here match what you would see if you called the OpenRouter models endpoint directly.

Frequently asked questions

How much does Nova Pro 1.0 cost per 1M tokens?

Nova Pro 1.0 costs $0.80 per 1M input tokens and $3.20 per 1M output tokens on OpenRouter. A typical chat run (1,000 input, 500 output) costs about $0.0024; a long-context job (50,000 input, 5,000 output) costs about $0.056.

What is the context window for Nova Pro 1.0?

The context window for Nova Pro 1.0 is 300,000 tokens, suitable for long documents and extended agent loops. Anything longer has to be chunked or summarized before it fits.

Does Nova Pro 1.0 support tool use / function calling?

OpenRouter does not publish a tool-use flag for Nova Pro 1.0 on this listing, so assume no and validate in your own integration. If the upstream feed adds the flag, the next six-hourly refresh will surface it here.

Is Nova Pro 1.0 cheaper than Nova Lite 1.0?

No — Nova Pro 1.0 is more expensive than Nova Lite 1.0 on input by about 92% ($0.80 vs $0.06 per 1M). Output is $3.20 per 1M versus $0.24 for Nova Lite 1.0. See the Nova Lite 1.0 model page for the full price table and workload examples.

How much does it cost to run Nova Pro 1.0 for chatbot?

A typical chatbot run (1,000 input + 500 output) on Nova Pro 1.0 costs about $0.0024 per call. At 10,000 runs/month that is roughly $24.00. The bulk of that is output at $3.20 per 1M tokens.
