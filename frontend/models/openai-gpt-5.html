About GPT-5

GPT-5 is priced in the premium band at $1.25 per 1M input tokens and $10.00 per 1M output tokens. It is aimed at workloads where the model is doing real work — long-form analysis, multi-turn agents, code reviews of nontrivial pull requests — and where cheaper models from the same provider would not produce a strong enough answer. The pricing reflects that role: not the most expensive offering, but not a budget pick either, and the right default for work that needs the provider's better-than-mid-tier capability.

At $1.25 per 1M input tokens and $10.00 per 1M output tokens, the cost of a single call is dominated by output unless the workload is very output-heavy or very input-heavy in unusual ways. Output is priced noticeably above input (8.0x), reflecting the typical chat-shape where the model spends more compute on generation than on reading the prompt. The context window is about 400K tokens, suitable for long documents and extended agent loops without having to chunk the input. OpenAI's pricing for this model is published through OpenRouter and refreshed every six hours; this page reads the live values, so the numbers here match what you would see if you called the OpenRouter models endpoint directly.

GPT-5 is the right pick when the answer quality is worth paying for. It fits well in: tasks that benefit from explicit chain-of-thought: math, code, multi-step planning, and structured analysis, agent loops that need to reason about tool choices and the order of operations before they act, long-form analytical writing where the model needs to work through a problem before producing the answer, and cases where the answer is high-stakes and the extra reasoning tokens are worth the price.

It is not the right pick for: high-volume work where the per-call cost would dominate the bill — for that, drop to a budget model in the same provider lineup and accept slightly weaker answers, or throwaway or experimental runs where the answer quality is not the point — cheaper models will produce similar results for those tasks. For those workloads, the right move is to pick a model from a different tier in the same provider lineup, or to use a different provider entirely.

When to use GPT-5

  • Math, code, and multi-step reasoning where chain-of-thought helps the answer
  • Agent loops that need to reason about tool choices before acting
  • Long-form analytical writing that benefits from explicit thinking
  • High-stakes answers where the extra reasoning tokens are worth the cost

When NOT to use GPT-5

  • Workloads where gpt-oss-120b (free) ($0.00/1M input) would produce a comparable answer at lower cost — premium tier is the right call when the quality gap is visible in your eval, not when it is theoretical.
  • Bulk extraction, classification, or routing pipelines where the per-call cost dominates the bill — a cheap-tier model from the same provider will handle the same workload at 5-10x lower cost than GPT-5.

Frequently asked questions

How much does GPT-5 cost per 1M tokens?

GPT-5 costs $1.25 per 1M input tokens and $10.00 per 1M output tokens on OpenRouter. A typical chat run (1,000 input, 500 output) costs about $0.0063; a long-context job (50,000 input, 5,000 output) costs about $0.113.

What is the context window for GPT-5?

The context window for GPT-5 is 400,000 tokens, suitable for long documents and extended agent loops. Anything longer has to be chunked or summarized before it fits.

Does GPT-5 support tool use / function calling?

OpenRouter does not publish a tool-use flag for GPT-5 on this listing, so assume no and validate in your own integration. If the upstream feed adds the flag, the next six-hourly refresh will surface it here.

Is GPT-5 cheaper than gpt-oss-120b (free)?

No — GPT-5 is more expensive than gpt-oss-120b (free) on input by about 100% ($1.25 vs $0.00 per 1M). Output is $10.00 per 1M versus $0.00 for gpt-oss-120b (free). See the gpt-oss-120b (free) model page for the full price table and workload examples.

How much does it cost to run GPT-5 for complex reasoning?

A typical complex reasoning run (2,000 input + 4,000 output) on GPT-5 costs about $0.0425 per call. At 10,000 runs/month that is roughly $425.00. The bulk of that is output at $10.00 per 1M tokens.
