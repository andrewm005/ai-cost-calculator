      Common questions

      How accurate are the prices?

      Live from OpenRouter's API, refreshed every 6 hours. Hand-curated prices are checked against each vendor's pricing page on every refresh. See the raw model list for the source of each entry.

      Do you store my prompts or my results?

      No. All math runs in your browser against a local API. There is no analytics, no tracking, no signup, no cookies set by this page. (Your browser may store localStorage for the optional favorites feature. Clear it any time from your browser settings.)

      What's the difference between input, output, cached, and reasoning tokens?

      Input is what you send the model. Output is what it sends back. Cached input is input you send repeatedly. Some providers (OpenAI, Anthropic) discount this ~50-90%. Reasoning tokens are the model's "thinking" output (o1, o3, Claude with extended thinking, Gemini 2.5 Thinking) and are billed at the same rate as output. All four are priced separately in the result.

      Can I compare two models side-by-side?

      Yes. Toggle to Compare above the model picker and pick 2-5 models. The same workload (project preset, task size, thinking) is priced against each, sorted cheapest first. Useful for the "should I use GPT-4o or Llama 3.3 70B" question.

      Why are some models missing from the list?

      OpenRouter catalogues around 336 models; the picker shows all of them. For open-weights models that aren't on OpenRouter yet, you can estimate the local self-host cost in the Also estimate local GPU cost in the panel below. it prices electricity + GPU time for a given Ollama model.

      Do I need to pay or sign up?

      No. Free to use, no account. The only optional sign-up is to OpenRouter itself if you decide to actually call the models through their unified API (which is the easiest way to use many of them with one key).
